Agents API

This section covers the core C++ agent classes.

BlueBoat API

class ABlueBoat : public AHolodeckBuoyantAgent

Surface vessel agent (BlueBoat) for multi-domain simulation.

  • Inherits from the HolodeckBuoyantAgent class. On any tick this object will apply the given forces. Desired values must be set by a controller.

Public Functions

ABlueBoat()

Default Constructor.

Sets default values for the BlueBoat agent.

  • Initializes the AI controller, physical properties (mass, volume, buoyancy center), and shifts the 2 thruster locations based on the offset to the origin.

virtual void InitializeAgent() override

Initializes the agent and its physical components.

  • Casts the RootComponent to a UStaticMeshComponent to allow physics manipulation and calls the parent class initialization.

virtual void Tick(float DeltaSeconds) override

Called each frame.

Called every frame to apply physical forces to the boat.

  • Converts linear and angular accelerations from the Python CommandArray into forces and torques, clamps them to maximum limits, and applies them to the RootMesh.

Parameters:

  • DeltaSeconds – The time elapsed since the last tick.

  • DeltaSeconds – The time since the last tick.

inline virtual unsigned int GetRawActionSizeInBytes() const override

Gets the size of the action buffer in bytes.

Returns:

The size of the command array in bytes (6 floats).

inline virtual void *GetRawActionBuffer() const override

Gets a pointer to the raw action buffer.

Returns:

A void pointer to the command array.

inline virtual float GetAccelerationLimit() override

Gets the acceleration limit for the agent.

  • Allows agent to fall up to ~8 meters.

Returns:

The acceleration limit.

void ApplyThrusters(float *const ThrusterArray)

Applies specific force values to the boat’s thrusters.

  • Iterates through the 2 thrusters, clamps the requested force to BB_MAX_THRUST, converts the force vector to Unreal Engine’s coordinate system, and applies it as a local force at the respective thruster locations.

Parameters:

  • ThrusterArray – Pointer to an array containing the requested force for each thruster.

  • ThrusterArray – Pointer to an array containing the requested force for each thruster.

void EnableDamping()

Enables linear and angular damping for the boat’s physical mesh.

  • For empty dynamics, damping is disabled. It should be enabled when using thrusters or a controller to simulate water resistance.

Public Members

TArray<FVector> thrusterLocations = {FVector(-250, -100, 0), FVector(-250, 100, 0)}

Location of all thrusters - Left and Right.

bool Perfect = true

Defines if the buoyancy calculation is perfect or approximated.

Sets default values for the DJI Matrice agent. Initializes the AI controller, ticking, and collision settings.

Initializes the agent and calculates the rotor arm length.

Initializes the UAV physical properties. Calculates gravity, mass, center of mass, and the lever arm length for torque calculations.

virtual void Tick(float DeltaSeconds) override

Called each frame.

Called every frame to apply linear and angular acceleration.

Parameters:

  • DeltaSeconds – The time since the last tick.

  • DeltaTime – The time elapsed since the last tick.

inline virtual unsigned int GetRawActionSizeInBytes() const override

Gets the size of the action buffer in bytes.

The size of the command array in bytes (6 floats).

inline virtual void *GetRawActionBuffer() const override

Gets a pointer to the raw action buffer.

Returns:

A void pointer to the command array.

void ApplyThrusters(float *const ThrusterArray)

Applies thrust to the 4 rotors based on a percentage array.

Applies thrust and calculates torque for the 4 drone rotors. Uses a thrust curve equation to map percentage to force, then calculates the resulting roll, pitch, and yaw torques based on the arm length.

Parameters:

  • ThrusterArray – Pointer to an array containing the requested percentage for each rotor.

  • ThrusterArray – Pointer to an array of requested percentages (0-100) for each rotor.

void ApplyFlightForce()

Applies aerodynamic drag and gravity to the UAV.

Applies drag and gravity to the UAV based on current velocity.
